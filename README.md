# hynix-stock-analysis & forecasting

네이버 뉴스에서 하이닉스 관련 뉴스를 자동으로 수집하는 크롤링 코드 및 감성분석을 통한 주가 예측

## 📊 프로젝트 개요

- **목적**: 하이닉스 관련 뉴스 데이터 수집을 통한 주가 예측분석
- **데이터 소스**: 네이버 뉴스
- **수집 기간**: 2024년 1월 1일 ~ 2025년 5월 31일
- **수집 데이터**: 뉴스 제목, 발행일, 미리보기 텍스트

## 기술 스택
- Python, PostegreSQL
- Flask
- HTML, CSS
- DBeaver
