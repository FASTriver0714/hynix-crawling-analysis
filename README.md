# 📈 SNS 감성분석을 이용한 주가 방향성 예측

> 네이버 주식토론방 게시글 데이터를 활용하여 SK하이닉스의 익일 주가 이동 방향 및 변동폭을 예측하는 딥러닝 모델 구축

## 프로젝트 개요

본 프로젝트는 Naver의 종목토론방의 게시글 분석이 주가의 이동 방향 및 변동폭 예측에 도움이 되지 않을까라는 생각에서 시작된 프로젝트로, 게시글은 직접 크롤링하여 수집하였으며 네이버 주식토론방에서 수집한 비정형 텍스트 데이터가 주가 변동에 대한 예측력을 가지는지 검증합니다.

참고한 논문(아래에 기재)은 시간 단위 주가 데이터를 사용하여 1시간 후 예측을 수행하였으나, 본 프로젝트에서는 **일 단위(daily) 주가 데이터**를 사용하여 **익일 예측**으로 실험을 재설계하였습니다. 이에 따른 샘플 수 제약을 보완하기 위해 베이스라인 모델 추가, 개별 게시글 단위 학습, 과적합 방지 설정 등의 개선을 적용하였습니다.

> **Note**: 본 프로젝트는 팀 프로젝트로 시작하였으나 비교적 미완성 상태에서 마무리된 후, 개인적으로 전체 파이프라인을 리팩토링하여 완성한 결과물입니다.

## 데이터

| 구분 | 설명 |
|------|------|
| 게시글 데이터 | 네이버 주식토론방 SK하이닉스 게시글 (2024~2025년) |
| 주가 데이터 | SK하이닉스 일별 OHLCV (2024~2025년, 거래일 기준) |
| 게시글 수 | 약 104,000건 (분 단위 timestamp 포함) |
| 거래일 수 | 약 338일 |

## 방법론

### 텍스트 전처리

원문 게시글에 대해 아래 순서로 전처리를 수행하였습니다.

1. 결측치 및 빈 문자열 제거
2. 광고글 제거 (동일 텍스트 3회 이상 반복)
3. 불용어·특수문자·URL 제거
4. KoNLPy Okt 형태소 분석기를 이용한 형태소 분리
5. Keras Tokenizer를 이용한 정수 인덱스 변환
6. Zero Padding (시퀀스 길이 통일)

### 종속변수 설계

| 변수 | 정의 | 설명 |
|------|------|------|
| r (주가 이동 방향) | close(t+1) ≥ close(t) → 1, else 0 | 익일 종가가 당일 종가 이상이면 상승 |
| R (주가 변동폭) | high(t+1) - low(t+1) ≥ δ → 1, else 0 | 익일 변동폭이 전체 기간 중간값 이상이면 고변동 |

### 독립변수-종속변수 매칭 (2가지 방식)

- **방식 A (날짜별 그룹 단위)**: 같은 거래일에 작성된 게시글을 하나의 그룹으로 묶어 익일 주가와 매칭 (338 샘플)
- **방식 B (개별 게시글 단위)**: 각 게시글을 독립 샘플로 사용하여 해당 거래일의 익일 주가와 매칭 (104,102 샘플)

### 모델 구조

**베이스라인 모델** (TF-IDF 기반)
- Logistic Regression
- Random Forest

**딥러닝 모델** (Embedding 기반)
- LSTM
- 1D CNN
- CNN + LSTM (Hybrid)

과적합 방지를 위해 원 논문 대비 dropout 상향(0.2→0.3), batch size 축소(128→64), 뉴런/필터 수 축소 범위 확대 등을 적용하였습니다.

## 주요 결과

### 최적 모델

| 방식 | 목표변수 | 모델 | 정확도 | p-value | 통계적 유의성 |
|------|----------|------|--------|---------|-------------|
| A | r (주가 방향) | CNN+LSTM (f=64, u=32) | **63.24%** | **0.019** | ✅ 유의 (p < 0.05) |
| A | R (변동폭) | CNN | 57.35% | - | ❌ |
| B | r (주가 방향) | - | 47.86% | - | ❌ |
| B | R (변동폭) | - | 48.23% | - | ❌ |

### 방식 A vs 방식 B 성능 비교

| 목표변수 | 방식 | 샘플수 | BL 최고 Acc | DL 최고 Acc | DL 최고 F1 | 통계유의(DL) |
|----------|------|--------|------------|------------|-----------|-------------|
| r | 방식A | 338 | 0.4706 | **0.6324** | 0.6753 | ✅ |
| r | 방식B | 104,102 | 0.4938 | 0.4786 | 0.6474 | ❌ |
| R | 방식A | 338 | 0.4559 | 0.5735 | 0.6506 | ❌ |
| R | 방식B | 104,102 | 0.4843 | 0.4823 | 0.6150 | ❌ |

### 핵심 발견

**1. 방식 A가 방식 B보다 우수함.**
샘플 수가 적음에도 불구하고 날짜별 그룹 단위(방식 A)가 더 높은 예측 정확도를 보였습니다. 개별 게시글 단위(방식 B)에서는 같은 날 게시글이 동일 레이블을 공유하면서 날짜 내 노이즈만 학습하는 한계가 나타났습니다.

**2. 주가 방향 예측에서 유의미한 결과를 확인하였습니다.**
방식 A + CNN+LSTM 조합에서 63.24%의 정확도(p=0.019)로 통계적으로 유의미하게 50%를 상회하는 예측력이 확인되었습니다.

**3. 딥러닝 모델에서 과적합이 주요 과제**
일 단위 데이터의 샘플 수 제약(338개)으로 인해 대부분의 딥러닝 모델에서 Train Accuracy는 높지만 Validation Accuracy가 정체되는 과적합 현상이 관찰되었습니다.

## 한계점 및 향후 과제

- **샘플 수 제약**: 일 단위 주가 데이터로 인해 방식 A 기준 338개의 샘플만 확보. 시간 단위 주가 데이터를 활용하면 원 논문과 동일한 실험 설계가 가능하며, 더 robust한 결과를 기대할 수 있음
- **과적합**: 대부분의 딥러닝 모델에서 과적합 발생. 더 강한 정규화 기법(L2, data augmentation 등)이나 사전학습 모델(KoBERT, KcELECTRA) 활용 검토 필요
- **단일 종목**: SK하이닉스 1개 종목만 대상으로 하여 일반화에 한계. 다양한 섹터의 종목으로 확장 필요
- **감성분석 고도화**: 현재는 형태소 단위 토큰화 후 End-to-End로 학습하였으나, 금융 도메인 특화 감성사전이나 사전학습 언어모델을 활용하면 성능 개선 가능성 있음
